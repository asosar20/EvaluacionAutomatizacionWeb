# Evaluación de Automatización Web

Este proyecto contiene pruebas automatizadas utilizando Cucumber y Java.

## Pre-requisitos
* Java JDK 17
* Maven configurado en las variables de entorno

## Ejecutar las pruebas
Para correr los tests y generar el reporte de evidencias, abre una terminal en la raíz del proyecto y ejecuta:

```bash
mvn clean test
